# Validation Mandate

Every coding task requires validation before you report DONE:

1. **Test** — Run the test suite. Every change must be exercised. No untested code ships.
2. **Audit** — Run the linter. Zero new errors. Fix errors regardless of when they were introduced.
3. **Verify** — Review the diff. No unintended changes. No scope creep. No leftover debug code.

Verification evidence is required. "Should work" is not verification.
